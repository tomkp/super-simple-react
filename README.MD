## super-simple-react


Install
```
npm i
```


Build
```
npm run build
```


Build & Watch
```
npm run build:watch
```

### Built with

 * **react** - js ui library
 * **babel** - compiler
 * **webpack** - packager / bundler
 * **postcss** - css processor
 * **normalize.css** - css 'reset'
 * **mocha** - testing
 * **enzyme** - testing React
 * **chai** - assertions
