## super-simple-react


Getting started with React


Install 
```npm i```

Build 
```npm run build```

Build & Watch 
```npm run build:watch```

Test 
```npm run test```

Test & Watch 
```npm run test:watch```

Open ```index.html``` in your favourite browser


---



 * **react** - js ui library
 * **babel** - compiler
 * **webpack** - packager / bundler
 * **postcss** - css processor
 * **normalize.css** - css 'reset'
 * **mocha** - testing
 * **enzyme** - testing React
 * **chai** - assertions
